# ETL Cotacao
Pipeline ETL que extrai dados de da cotacao das moedas: Dolar, BTC e Euro para Real para a data da execucao via uma api publica, 
transforma em DataFrame e carrega em Excel.
## Tecnologias
- Python
- Pandas
- Requests
## Como executar
1. Instalar dependências: pip install -r requirements.txt
2. Rodar: python src/[main.py](http://main.py)
